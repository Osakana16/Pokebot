# Pokebot
This is a bot plugin for Counter-Strike(1.6).

Work in progress.

## What it can be done
The bot can reach the goal and accomplishes the objectives.

Combat process is incomplete.

## Features
- Behavior based
- Waypoint navigation

## Goals
- Make bots fight.
- Make bots team up.
